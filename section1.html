Lecture 01

Geminiで校務改革

生成AIで「書く・まとめる・伝える」をもっと楽に

話した内容を、相手に伝わる文章へ。

ねらい

日々の校務にある「話す・書く・まとめる」仕事を、Geminiとの対話で軽くします。スマートフォンで録音・文字起こしした内容をGeminiに貼り付け、同じ内容から、校内向け議事録、保護者向け連絡文、管理職向け報告、自分用メモを作り分ける方法を体験します。

キーワード

音声の文字起こし / Gemini / 校務文書 / プロンプト / 個人情報保護

使用するツール

  • スマートフォン
  • 標準の録音アプリ
  • Gemini
  • Webブラウザ

事前準備

  • radio_button_uncheckedGoogleアカウントへログインできる状態
  • radio_button_uncheckedスマートフォン
  • radio_button_unchecked可能であれば、個人情報を除いた2〜3分程度の文字起こし
  • radio_button_unchecked文字起こしがない場合は当日用サンプルを使用可能

端末別のおすすめ運用

iPhone

ボイスメモ → 文字起こし → テキストをコピー → Geminiへ貼り付け

Pixel

Googleレコーダー → 文字起こし → テキストをコピー → Geminiへ貼り付け

その他のAndroid

標準の録音アプリ → 文字起こし機能があれば利用 → テキストをコピー → Geminiへ貼り付け

補足: 端末によって録音・文字起こしの方法は異なりますが、文字になった後の流れは共通です。

warning

個人情報を含む資料を生成AIへ入力しないでください。児童生徒名、保護者名、住所、電話番号、成績、健康情報などは、番号・記号・仮名に置き換えてください。AIが生成した内容は、そのまま使用せず、必ず教職員が内容を確認してください。

コピーして使えるプロンプト

1. 校内向け議事録

会議・教育相談の文字起こしを、共有用の議事録に整理します。

以下は、会議または教育相談の文字起こしです。 内容を校内で共有する議事録として整理してください。 次の項目に分けてください。 1. 話し合った内容 2. 決定事項 3. 確認事項 4. 担当者 5. 期限 6. 今後の対応 文字起こしに含まれていない情報を推測して追加しないでください。 【ここに文字起こしを貼り付けます】

2. 保護者向け連絡文

面談・教育相談の内容から、保護者へ送る連絡文を作成します。

以下は、面談または教育相談の文字起こしです。 内容をもとに、保護者へ送る連絡文を作成してください。 条件: ・専門用語を避ける ・安心感が伝わる表現にする ・学校と家庭が協力して支援する姿勢を示す ・断定的な表現を避ける ・300字程度にまとめる ・個人情報や不要な内容は記載しない 【ここに文字起こしを貼り付けます】

3. 管理職向け報告

校長・教頭へ報告するための簡潔な文書を作成します。

以下は、会議または教育相談の文字起こしです。 校長・教頭へ報告するための文書として、簡潔に整理してください。 次の項目に分けてください。 1. 概要 2. 現在の状況 3. これまでの対応 4. 今後必要な対応 5. 管理職に確認・判断してほしいこと 事実と推測を分け、文字起こしにない情報は追加しないでください。 【ここに文字起こしを貼り付けます】

事実と推測を分けて記載するよう、生成後に必ず確認してください。

4. 自分用メモ

担任が後日見返すための、短い振り返りメモを作ります。

以下の文字起こしを、担任が後日見返すためのメモにしてください。 次の内容を箇条書きで整理してください。 ・重要な発言 ・確認したこと ・今後行うこと ・次回確認すること ・期限があるもの 短く、すぐに読み返せる形にしてください。 【ここに文字起こしを貼り付けます】
  • ・学級通信作成
  • ・通知文作成
  • ・校内研修の要約
  • ・PTA会議録
  • ・進路面談記録
